# Document Archive

This repository now contains OpenDocument originals in [`files/`](./files/) plus browser-friendly preview copies in [`previews/html/`](./previews/html/) and [`previews/pdf/`](./previews/pdf/).

## Best Way To View

- On GitHub Pages from a phone: use [`index.html`](./index.html) or the files in [`previews/html/`](./previews/html/).
- In GitHub's repository file view: PDFs in [`previews/pdf/`](./previews/pdf/) are the most reliable inline preview format.
- The original `.ods` and `.odt` files in [`files/`](./files/) are kept for download and editing, but GitHub and GitHub Pages will usually treat them as files to open externally or download.
